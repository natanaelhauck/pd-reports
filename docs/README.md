# Documentação PD Reports

Este diretório reúne a documentação operacional e os materiais de apoio do projeto.

- `SETUP_LOCAL.md`: configuração para desenvolvimento local.
- `ATUALIZACAO_CONSUMO.md`: fluxo operacional de atualização manual do Consumo.
- `PERMISSOES_CIDADE.md`: regras de acesso por escopo municipal.
- `CHECKER_API_CERTIFICADOS.md`: referência do fluxo de certificados/checker.
- `AUDITORIA_PROJETO.md`: avaliação técnica, riscos e próximos passos.
- `PRIVACIDADE_DADOS.md`: política de dados, screenshots e arquivos sensíveis no repositório.
- `images/`: screenshots usados no README principal.

Antes de publicar imagens em portfólio, confirme que nomes, e-mails, matrículas, telefones, datas de nascimento, patrimônio e monitores estejam anonimizados.
